h1. English

TouchCustoms is a Cocoa Touch static library that contains a number of reusable components that we use in our projects on a daily basis.
* See "Wiki":http://wiki.github.com/eisernWolf/TouchCustoms/ for documentation.
* "*F.A.Q.*":http://wiki.github.com/eisernWolf/TouchCustoms/faq
* "License":http://github.com/eisernWolf/TouchCustoms/blob/master/LICENSE

h2. Component list

# @SCMemoryManagement@ - Make memory management easier.
# @SCRatingView@ - 5-star rating-view that allows to set ratings yourself.
# @SCSegmentedControl@ - Multi-row @UISegmentedControl@.
# @SCSymbolicProgressBar@ - A progress-bar that uses a series of identical symbols to represent the value of progress.
# @NSArray (Linq)@ - Category that adds some .NET LINQ abilities to Objective C. "Explanation on how to use that in your projects":http://screencustoms.com/post/933566043/a-little-bit-of-linq-in-objective-c

h2. Сьпіс кампанэнтаў

# @SCMemoryManagement@ - Спрашчае кіраваньне памяцяй.
# @SCRatingView@ - 5-ці-зоркавы рэйтынґ-вью, што дазваляе ўсталёўваць рэйтынґ карыстальніку.
# @SCSegmentedControl@ - Шмат-радковы @UISegmentedControl@.
# @SCSymbolicProgressBar@ - Проґрэс-бар, які выкарыстоўвае шэраг аднолькавых відарысаў дзеля адлюстраваньня статусу.
# @NSArray (Linq)@ - Катэґорыя, якая дадае у Objective C штосьці з мажлівасьцяў .NET LINQ. "Тлумачэньне па выкарыстаньні ў вашых праектах":http://screencustoms.com/post/933566043/a-little-bit-of-linq-in-objective-c (на анґельскай)
